# 🌸 Florescer — Floricultura Online

Landing page premium para floricultura online, desenvolvida com HTML, CSS e JavaScript puro. O projeto apresenta uma identidade visual sofisticada, animações fluidas e layout totalmente responsivo.

## ✨ Funcionalidades

- **Cursor personalizado** com animação suave de seguimento
- **Hero section** com grid de imagens escalonado
- **Contador animado** nas estatísticas ao entrar na tela
- **Scroll reveal** — seções animam ao aparecer na viewport
- **Notificação de carrinho** com feedback visual
- **Newsletter** com confirmação animada
- **Navbar** com efeito de encolhimento ao rolar a página
- **Layout responsivo** para mobile e desktop

---

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações, `clamp()`)
- JavaScript vanilla (ES6+)
- Google Fonts — Cormorant Garamond + DM Sans
